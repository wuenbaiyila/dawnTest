一、             系统亮点

1、 面试公司：

1) 、本系统在同学们的努力下收集了北京几百家软件公司的面试题和笔试题

2) 、当你收到面试邀请后,通过本系统搜索到该公司的笔试题和面试题,提前做好准备找到答案,让你快速的 万薪就业.

3) 、面试公司搜索方法.



2、 知识点收集

1) 、本系统针对所学专业的知识点进行了分类,让你更快速的找到你要学习的重点

2) 、系统分为h5移动开发专业,UI设计专业,软件测试专业,Java大数据专业,我们还根据每个专业进行了精细 化的知识点划分

3、 如何利用好此功能

1) 、此功能可以在学习过程中,学完一个知识点就可以来这里看下,把该知识点下的所有问题全背会.

2) 、知识点点击量,这个数据是大家出去面试完回来维护的,只要有公司提问到了该知识点数量就会累加
也就是说提问次数越多的题我们越要背的牢固.



二、             系统角色划分

1、教师角色：

1) 导入用户权限



2) 教师角色拥有的权限是用红色框标出的

3) 维护学生信息权限



4) 审核学生提交的面试总结权限,学生每天提交的面试题老师都可以在这里审核到
先查看学生提交的面试总结是否符合要求



2、学生角色：

1) 提交面试总结,每天回来提交面试总结,首先要搜索系统中是否有该公司,系统里有该公司的情况,直接在公司里添加面试总结就可以



2) ,系统里没有该公司的情况,直接到自己所学的专业的公司导航菜单里添加,例如H5专业




3) ,我的面试总结老师未审核的题目都在这里可以看到,审核通过后才可以在公司里显示
